here# SS1 Chemistry — First Term Notes

Welcome to the **SS1 Chemistry First Term Notes** repository! This project contains clear, mobile-friendly revision notes for SS1 students.

## Features

- Covers all **first term chemistry topics** for SS1
- Easy to read and **mobile-friendly**
- **Searchable topics** for quick reference
- Can be **opened offline** in a browser
- Includes **simple experiments and examples**

## Topics Included

1. What is Chemistry?
2. States of Matter
3. Classification of Matter
4. Elements and Symbols
5. Mixtures & Separation Techniques
6. Laboratory Apparatus & Their Uses
7. Measurement & SI Units
8. Atoms — Basic Structure
9. Chemical Formulae & Simple Equations
10. Simple Experiments & Observations

## How to Use

1. Open the website in a browser (works on mobile and desktop).
2. Use the search box to quickly find topics.
3. Tap on a topic to expand and read the notes.
4. Copy notes or print them as needed.
